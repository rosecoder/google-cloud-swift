# GCPDatastore

A Swift package as client for [Google Cloud Datastore](https://cloud.google.com/datastore).

## Features

| Feature                    | Implemented |
|:---------------------------|:------------|
| Namespaces                 | ✅          |
| Get                        | ✅          |
| Put                        | ✅          |
| Delete                     | ✅          |
| Allocate/Reserve IDs       | ❌          |
| Transactions               | ❌          |
| Query                      | ✅          |
| Query - Cursors            | ❌          |
| Query - Projection         | ❌          |
| Query - Distinct on        | ❌          |
| GQL                        | ❌          |
| GCP - Traces               | ❌          |
